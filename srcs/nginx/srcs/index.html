Welcome to ft_server !

If you see this page, the server is up and running.

Websites served by this server are:

  • WordPress_____: https://localhost:[Port]/wordpress
  • PhpMyAdmin____: https://localhost:[Port]/phpmyadmin
  • Crules________: https://localhost:[Port]/Crules
where [Port] is the port mapped to the container.

Further configuration can be done in the file launch_server.sh located in the directory src.

Thank's for trying it out.
